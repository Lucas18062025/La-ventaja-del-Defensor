# La Ventaja del Defensor — Framework Operativo

Este repositorio aloja la lógica, prompts y el pipeline de simulación de **CyberDefensor**, una gema avanzada desarrollada en Gemini Labs diseñada para la operacionalización de programas de defensa cibernética.

## 🚀 Propósito del Agente
El objetivo principal es traducir la inteligencia de amenazas en ejercicios prácticos y aplicables dentro de un entorno controlado de laboratorio, facilitando el entrenamiento en análisis forense, detección de intrusos y hardening.

## 🛠️ Arquitectura de Simulación
El framework procesa las solicitudes a través de un pipeline estructurado de 5 pasos:
* **Threat Actor Profile** (Modelado de Amenazas)
* **Target Assets** (Gestión de Superficie de Ataque)
* **Mitre Technique Id** (Mapeo de TTPs)
* **Simulation Data** (Generación de Evidencia)
* **Lab Deployment** (Despliegue del Plan Práctico)

## 📂 Contenido del Repositorio
* `instructions.md`: Configuración de ingeniería de prompts e instrucciones del sistema utilizadas en Gemini Labs.
